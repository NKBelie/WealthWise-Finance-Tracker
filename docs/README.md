## 🎯 Project Overview

### What is WealthWise?
WealthWise is a comprehensive personal finance management application built using the PERN stack (PostgreSQL, Express.js, React, Node.js). It's developed as part of the Codveda Full-Stack Development Internship program.

### Key Features
- **User Management**: Secure registration, authentication, and profile management
- **Financial Tracking**: Income and expense tracking with categorization
- **Budget Management**: Create, monitor, and analyze budgets
- **Real-time Updates**: Live notifications using Socket.io
- **Analytics**: Financial insights and trend analysis
- **Modern APIs**: Both REST and GraphQL implementations

### Technology Stack
```
Frontend:  React 18 + Vite + Tailwind CSS + Chart.js
Backend:   Node.js + Express.js + JWT Authentication
Database:  PostgreSQL 15+ with optimized queries
Real-time: Socket.io for live updates
GraphQL:   Apollo Server implementation
```
